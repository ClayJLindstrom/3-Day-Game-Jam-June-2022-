King Core

Game Jam -- June 2022


The Game

King Core is a Parkour Video Game. Climb up the wall of the castle to save the heir to the kingdom. The farther you climb, the higher your score.

Game Jam

What is a Game Jam

A Game Jam is a 3 day game developing context. On day one theme of the game is announced. This Jam's theme is Parkour that has to be score based.

For more information see the Game Jam 19 website
